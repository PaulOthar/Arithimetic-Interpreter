Arithimetic Interpreter Documentation

Made By: Paul / Github: PaulOthar



Quick Description:

This Arithimetic Interpreter is a Simple Algorythm That is Able To Read a String Containing an Equation, and Transforms it into a "Node" Object, Able To Solve To Acquire The Desired Result



How To Use:

For Your Convenience, i made a Class With Static Methods that can be Called To Run The Process Witheout The Need for Instancing Stuff.


Methods:

(Returner.BuildTokensFromEquation):

Recives a Equation (String) and a Array Of Variables (Variable).
Returns a Array of Tokens (Token), That Can Be Later Turned Into a Equation.

(Returner.BuildNodeFromTokens):

Recives a Array of Tokens (Token).
Returns a Single Node (Node), That Can Be Ran With The Inner Method (.solve()) Containing All The Other Nodes Necessary To Complete The Execution of The Equation.

(Returner.BuildNodeFromEquation):

Recives a Equation (String) and a Array Of Variables (Variable).
Returns a Single Node (Node), That Can Be Ran With The Inner Method (.solve()) Containing All The Other Nodes Necessary To Complete The Execution of The Equation.

Disclaimer:

1:Even a Single Number is Considered an Equation.

2:Every Result Yields a Array.

3:Operations Including Multiple Values Will Be Treated as Each of One Equation Numbers By Each of Other Equation Numbers.
Example 1:
[1,2,3] + 5
[1+5 , 2+5 , 3+5]
[6,7,8]

Example 2:
[0,1,2] - [3,4,5]
[0-3 , 0-4 , 0-5 , 1-3 , 1-4 , 1-5 , 2-3 , 2-4 , 2-5]
[-3,-4,-5,-2,-3,-4,-1,-2,-3]

Set Left :
Sets The Value Of The Left Variable To The Value On The Right, And Returns The Value (Just Returns The Value if it is not a Variable)
Recives : " (Variable) <- (Equation) "
Returns : " (Variable) " or " (Equation) "

Example:
Input: (X : 0) <- 3
Output: (X : 3)


Set Right :
Sets The Value Of The Right Variable To The Value On The Left, And Returns The Value (Just Returns The Value if it is not a Variable)
Recives : " (Equation) -> (Variable)"
Returns : " (Variable) " or " (Equation) "

Example:
Input: 3 -> (X : 0)
Output: (X : 3)


Repeat :
Repeats The Inside Node Multiple Times
Recives : " repeat{ (Equation) : (Equation) } "
Returns : " [(Result),(Result),(Result)...] "

Example:
Input: repeat{ 5 : ( (X : 0) + 1 -> (X : 0) ) }
Output: [1,2,3,4,5]


Ignore :
Runs The Inside Node but Returns 0 Anyway
Recives : " ignore{ (Equation) } "
Returns : " 0 "

Example:
Input: ignore{ X <- X+1 }
Output: 0 (But X Recives The Change)

Function :
